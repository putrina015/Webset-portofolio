Proyek Saya

KODINGAN WEB PORTOFOLIO

Kodingan web portofolio adalah kumpulan kode yang dirancang untuk menampilkan informasi pribadi, keterampilan, dan karya seseorang secara online.

MEMBUAT WEBSET PENJUALAN

Website penjualan adalah platform online untuk menjual produk. Proses pembuatannya meliputi perencanaan, desain tampilan menarik, pengembangan menggunakan HTML, CSS, dan JavaScript untuk frontend, serta PHP untuk backend.

WEB APPLICATION PROJECT

Web Application Project adalah sebuah proyek pengembangan aplikasi berbasis web yang dirancang untuk memberikan solusi digital yang dapat diakses melalui browser. Aplikasi ini biasanya dikembangkan menggunakan teknologi seperti HTML, CSS, JavaScript, dan bahasa pemrograman back-end seperti PHP, Python, atau Node.js. Selengkapnya
